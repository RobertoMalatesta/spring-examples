# How to run local
1. Clone the repository
2. Run the following command:
```shell
gradlew bootRun
```
3. Open your browser and navigate to `http://localhost:8080/hello` <- RestClient
4. Open your browser and navigate to `http://localhost:8080/hello2` <- RestTemplate
